# ALT 브랜드 브리프 (Claude Design 용)

> 이 문서는 **Claude Design/앱에 그대로 붙여넣는** 재사용 브랜드 가이드입니다.
> Claude Design은 브랜드 템플릿 파일 업로드를 지원하지 않으므로, 이 브리프가
> "통일된 서식"을 유도하는 핵심 장치입니다. `tokens.yaml` 이 갱신되면 이 문서의
> 수치도 함께 맞추세요. (아래는 채우기 전 자리표시 값 — 실제 브랜드로 교체)

## 1. 브랜드 톤
- ALT Consulting: 신뢰감 있고 정제된, 데이터 중심의 컨설팅 톤.
- 여백을 넉넉히, 한 슬라이드에 한 메시지. 장식보다 명료함 우선.

## 2. 색 팔레트 (이 색만 사용)
- Primary `#1A2B4A` — 표지 배경, 섹션 제목
- Secondary `#2E6BE6` — 포인트, 링크, 강조 도형
- Accent `#F5A623` — 핵심 수치/하이라이트에만 소량
- Text `#1C1C1E` / Muted `#6B7280` — 본문/보조 텍스트
- Background `#FFFFFF` / Surface `#F4F6F9` — 배경/카드

## 3. 타이포그래피
- 제목: **Pretendard**, 본문: **Pretendard**, 코드/수치표: JetBrains Mono
- 크기(pt): 표지 40 · 섹션 32 · 소제목 24 · 본문 16 · 캡션 12
- 굵기 대비로 위계 표현, 임의 폰트 혼용 금지.

## 4. 레이아웃 규칙
- 비율 16:9. 여백 상/하 48pt, 좌/우 56pt. 12컬럼 그리드.
- 좌상단 로고, 우하단 페이지 번호를 모든 본문 슬라이드에 배치.
- 지원 레이아웃: 표지 / 섹션 구분 / 본문 / 2단 / 이미지+텍스트 / 전면 이미지 /
  차트 / 데이터 표 / 인용.

## 5. 이미지
- 원본 비율 유지, 왜곡·과도한 크롭 금지.
- 사진은 통일된 톤(채도/명도), 아이콘은 단색 라인 스타일.

## 6. 차트/표
- 색은 위 팔레트만 사용, 데이터 잉크 최소화(불필요한 격자/테두리 제거).
- 핵심 수치는 Accent 로 강조.

## 7. 금지
- 팔레트 외 색, 지정 외 폰트, 그림자/그라데이션 남용, 저해상 이미지 확대.
